Goals & Audience

Build a simple, responsive personal portfolio to share with instructors, classmates, and future internship contacts.

Information Architecture

  • Home
  • About
  • Projects
  • Skills
  • Experience
  • Contact (mailto link)

Visual Design

Dark background with light text and a blue accent for links/buttons. System font stack for performance. Rounded cards separate content.

Wireframes / Mockups

Two low‑fi wireframes sketched before coding to plan layout and hierarchy.

Accessibility

  • Semantic HTML with headings and landmarks
  • Labels and clear link text where needed
  • High color contrast for dark theme
  • Keyboard-friendly navigation

Responsiveness

Single column on small screens and a 2‑column grid for project cards on wider screens. All images scale to container width.

Challenges & Resolutions

  • Scope vs. simplicity: Kept layout minimal to pass validation.
  • Contact Form: Had issues with the layout of my contact form but was able to resolve.
  • Error Code: Had issues with code not working due to opening and closing tags but after a deep look into every line I was able to fix all code.
  • Layout: Although layout was very minimal with complexity it took me a very long time to make sure everything was placed accordingly.
